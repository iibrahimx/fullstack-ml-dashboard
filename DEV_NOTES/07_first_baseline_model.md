# First Baseline Model

After exploring the dataset we trained the first baseline model.

## Why a baseline model?

A baseline model gives us a simple reference performance.

Before trying complex algorithms we need to know:

"How well can a simple model perform?"

## Model used

Logistic Regression

Reasons:

- simple
- interpretable
- good baseline for classification

## Train / Test split

We split the dataset:

80% training  
20% testing

This simulates real-world prediction where the model sees new unseen data.

## Evaluation metrics

Accuracy was used as the first metric.

Later we will also examine:

- confusion matrix
- precision
- recall
