# Dataset Overview

## Files included in the dataset

The Kaggle dataset contained three CSV files:

- data_01.csv
- data_02.csv
- data_for_training.csv

The main dataset used for model training is:

ml/data/data_for_training.csv

## Dataset shape

Rows: ~2000  
Columns: 9

## Features

monthly_income  
monthly_expense  
age  
has_smartphone  
has_wallet  
avg_wallet_balance  
on_time_payment_ratio  
num_loans_taken

## Target

target

The target column represents whether a user defaults on a loan.

This makes the problem a **binary classification task**.

## Why this dataset is good for learning

- Small and manageable
- Clean numeric features
- Realistic financial risk prediction scenario
- Works well with classic ML models like logistic regression or random forest
